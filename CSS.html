1. CLASS 선택자

적용 방법 : 1. 헤드 안에 .클래스명, 2. html 내 클래스 선언

2.ID 선택자

적용 방법 : 1. 헤드 안에 #클래스명, 2. html 내 ID속성 선언 차이점 : 1. ID속성은 문서내에 유일한 요소여야 함, 2. 구체성

3. 선택자의 조합

3.1. 요소+클래스 ex)p.bar 3.2. 다중 클래스 ex).foo.bar 3.3. ID + 클래스 ex) #foo.bar

4. 속성 선택자

4.1. 단순 속성값 ex) p[class][id] 4.2. 정확한 속성값 ex) p[class=""] 4.3. 부분 속성값
    4.3.1. [class~="bar"] : 공백으로 구분 요소 선택 4.3.2. [class^="bar"] : bar로 시작하는 요소 선택 4.3.2. [class$="bar"] : bar로 시작하는 요소 선택 4.3.2. [class*="bar"] : bar를 포함 요소 선택
5. 문서 구조 관련 선택자
    5.1. 자손 선택자 : 선택자와 선택자 사이 공백으로 표현 : ex) div span {} 5.2. 자식 선택자 : 선택자 > 선택자 : ex) div > span {} 5.3 인접 형제 선택자 : 선택자 + 선택자 ex) div + span {} 5.4 조합해서 사용 가능 : ex) body > div table + ul{}
6. 가상 선택자 6.1 가상 클래스명(:pseudo-class) : 미리 정의해놓은 상황(시나리오 or 조건)에 적용이 되도록 약속되어있는 보이지 않는 클래스 * 가상 클래스 색인 6.1.1. 문서 구조 관련 색인 :first-child : 첫 번째 자식 요소 선택 :last-child : 마지막 자식 요소 선택 6.1.2. 링크 관련 가상 클래스명 :link : 하이퍼링크이면서 아직 방문하지 않은 앵커 :visited : 이미 방문한 하이퍼링크 6.1.3. 사용자 동작 관련 가상 클래스 :focus : 현재 입력 포커스를 갖고 있는 요소에 적용 :hover : 마우스 포인트가 위치해 있는 요소에 적용 :active : 사용자 입력에 의해 활성화된 요소에 적용 6.1.4.그 밖의 가상 클래스 색인들 https://www.w3schools.com/css/css_pseudo_classes.asp 6.2 가상 요소 : 미리 정의해놓은 위치에 삽입이 되도록 약속되어있는 보이지 않는 요소 6.2.1. 대표적인 요소 :before : 가장 앞에 요소를 삽입 :after : 가장 뒤에 요소를 삽입 :first-line : 요소의 첫 번째 줄에 있는 텍스트 :first-letter : 블록 레벨 요소의 첫 번째 문자 6.2.2. 그 밖의 가상 요소 https://developer.mozilla.org/en-US/docs/Web/CSS/content 7. 구체성
