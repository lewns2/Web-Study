자주사용하는 태그

1.제목과 단락

h1~h6 : 헤딩 태그, p : 단락 태그, br : 줄바꿈 태그

2.텍스트 관련 태그(Inline text semantics)

b : 굵게, i : 기울기체 , u: 밑줄, s : 중간선

3.앵커 요소(The Anchor element)

앵커(a) : 다른 문서로 이동할 수 있는 링크 생성, href : 링크

href="#ID속성" : 내부에서 사용 시

기타속성 : 참고사이트

4.의미없는 요소를 묶기 위한 태그(Container)

  • 목적 : 서버에서 주는 데이터를 담기 위함, 스타일을 주기 위함
    • div : block-level
    span : inline-level

    5.리스트

    ul : 순서가 없는 리스트(순서와 상관없을 때)
    ol : 순서가 있는 리스트
    dl : 용어 설명 리스트, dt : 용어를 구분, dd : 용어의 정의

    6.이미지 요소

    img : 문서에 이미지 삽입, src: 이미지 경로 지정, alt : 이미지 대체 텍스트 입력, width/height : 이미지 크기
